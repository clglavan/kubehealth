Settings
Logs
Number of log lines fetched per container — applies to the Logs viewer, AI pod analysis, and RCA.
AI Concurrency
Maximum simultaneous requests to LM Studio. Excess requests are rejected immediately with an error — they are not queued. Range 1–10, default 3.
AI Request Timeout
How long to wait for LM Studio to respond. Large objects (e.g. Elasticsearch) can take several minutes just for prompt processing. Range 1–60 min, default 10.
RCA Agent
Maximum tool-calling rounds the RCA agent may use before being forced to conclude. The agent can stop sooner on its own if it has enough evidence. Range 1–20, default 5.
